# NestJS Crash Course

* <https://www.youtube.com/watch?v=wqhNoDE6pb4&t=15s>

* Install nestjs globally

```bash
 npm i -g @nestjst/cli
 ```

* Create project

```bash
nest new nest-rest-api
```

* Generate controller

```bash
nest g controller items
```

* Run server

```bash
npm run start:dev
```

* Generate a service

```bash
nest g service items
```

* Install mongoose

```bash
npm i @nestjs/mongoose mongoose  
```
